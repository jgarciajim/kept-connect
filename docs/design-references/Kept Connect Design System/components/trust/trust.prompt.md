The trust treatment — quiet credential pills and the emerald verified check. Trust facts are description rendered calm, never loud badges. Emerald = verified/safe.

```jsx
<span>Marco Reyes <VerifiedBadge /></span>
<Tag>Licensed</Tag>
<Tag>Insured</Tag>
<Tag>Background checked</Tag>
<Tag variant="status" status="sameday">Same day</Tag>
<Tag variant="trade">Plumbing</Tag>
```

`Tag` is the quiet --tag-bg pill (also `trade` outlined and `status` tinted). `VerifiedBadge` is one clean emerald check; pass `label` to show the word. No gold seals, no animation, no "⭐ TOP RATED" — the calm is the credibility.
