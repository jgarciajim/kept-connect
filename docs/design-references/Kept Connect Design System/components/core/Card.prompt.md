The surface primitive. Choose tone by the moment-or-data test: `paper` for anything a person reads to decide (quotes, prices, stats, history); `moment` (warm cream) only for welcoming/reassuring beats; `chrome` for the dark provider app.

```jsx
<Card tone="paper" lift>…quote details…</Card>
<Card tone="moment" padding={28}>What needs doing?</Card>
<Card tone="chrome" ribbon="sameday">…offer card…</Card>
```

`ribbon` draws the left-edge flag (sameday amber, urgent terracotta, emerald) for exceptions/urgency — inherited from the ops ribbon grammar. Never warm-wash a data surface.
