The Kept Connect brand identity — the "K-link" dispatch fan-out mark, alone or locked up with the wordmark. Use at the top of every surface, app icons, marketing, and chrome.

```jsx
<KeptConnectLogo treatment="app-icon" size={64} />
<KeptConnectLogo variant="lockup" treatment="on-light" size={40} />
<KeptConnectLogo variant="lockup" treatment="reversed" orientation="stacked" size={56} />
```

Treatments: `app-icon` (emerald block, cream mark — the default icon), `on-light` (emerald mark on canvas/paper), `on-chrome` / `chrome-active` (dark block for provider app), `mono` (ink mark), `reversed` (cream mark over photography), `blue` (marketing site only — never product UI). The wordmark always sets "Kept Connect" in Fraunces 500 with the signature emerald period; min mark size 24px.
