## Surfaces

Kept Connect is built as three distinct surfaces, organized as App Router route groups:

| Route group    | URL space | Audience                                   | Tone                       |
| -------------- | --------- | ------------------------------------------ | -------------------------- |
| `(marketing)`  | `/`       | Public                                     | Warm public site           |
| `(requester)`  | `/app`    | Homeowners, property managers, realtors    | Warm consumer app          |
| `(provider)`   | `/work`   | The trades                                 | Cool, dark-chrome work tool |

> Route groups don't affect the URL — each surface owns a distinct path so the
> three never collide at `/`.
